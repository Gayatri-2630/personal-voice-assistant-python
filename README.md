# Personal Voice Assistant using Python

## Overview
This project implements a simple voice assistant using Python that can recognize speech, respond with voice output, and perform tasks like playing songs, telling time, fetching Wikipedia info, and telling jokes.

## Features
- Speech recognition
- Text-to-speech responses
- Play songs on YouTube
- Tell current time
- Fetch information from Wikipedia
- Tell jokes

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia API

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python voice_assistant.py
